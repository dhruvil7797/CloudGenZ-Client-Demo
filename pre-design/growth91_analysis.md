## 1. Brand Audit

### Current Site Summary
Growth91 is India's leading startup investment platform connecting retail investors with high-growth, pre-vetted Indian startups. The platform serves both **investors** (who want to discover deals) and **founders** (who want to raise capital).

### Current Visual Identity
| Attribute | Current Value |
|-----------|--------------|
| Primary Color | Deep Navy/Indigo `#1a1069` |
| Accent Color | Vibrant Orange `#f97316` |
| Background | Dark navy gradient `#12106e → #1a1a8c` |
| Typography | System Sans-serif (generic) |
| Logo | Leaf/Sprout icon + "Growth91" wordmark |
| Hero | 3-slide carousel with growth chart illustration |

### Current Page Structure
1. **Nav** — Logo, Home, Deals, Founders, Resources, About Us, Blog, Future Unicorns (NEW badge), Sign In, GET STARTED CTA
2. **Hero** — 3-slide: Invest in Startups / For Founders / Raise Funds
3. **About Section** — "Discover, Trust, Invest" with numbered accordion + illustration
4. **Social Proof** — "Startups liked by Marquee Angels" logo wall
5. **Stats** — 75+ funded, 90% success rate, 1000+ analyzed, Top Quartile
6. **Why Invest** — 3 feature cards (Transparency, Handpicked Deals, Aligned Interests)
7. **Testimonials** — Carousel with 3 investor testimonials + ratings
8. **Blog** — 3 investment guide articles
9. **FAQ** — 4 accordion questions
10. **CTA Banner** — "Join Us to Invest in Startups"
11. **Footer** — Minimal, copyright + social icons

### Brand Weaknesses (Current Site)
- Carousel hero feels dated; no visual depth
- Inconsistent illustration style (cartoon icons)
- Stats section lacks visual hierarchy
- No clear investor journey/pathway visualization
- Missing deal flow preview / live portfolio cards
- No trust indicators beyond testimonials
- Footer is too minimal
- No "How It Works" process flow
- Missing sector/stage filter showcase
- No community/network visualization

## 2. Competitor Intelligence

### OpenVC (openvc.app)
- **Style:** Clean white, hot pink CTA, data-table UI
- **Key Components:** Investor search with live filtering, social proof banner (40,000+ founders), media mentions row
- **Takeaway:** Data-first UX, investor database preview builds trust immediately
- **Steal:** Live investor count in hero, searchable database preview, "As seen in" media strip

### Vestbee (vestbee.com)
- **Style:** Light blue/white, rounded cards, professional
- **Key Components:** Dual-audience split (Startups/Investors), partner logos (AWS, HubSpot), role-based cards (Accelerators, Angels, Corporations)
- **Takeaway:** Role-segmented CTAs are powerful for multi-audience platforms
- **Steal:** Bento-style role cards, ecosystem partner logos, split audience hero

### TykeInvest (tykeinvest.com)
- **Style:** Dark, fintech premium
- **Key Components:** Portfolio returns visualization, deal cards, SEBI compliance badge
- **Takeaway:** Showing actual returns/deals creates instant credibility
- **Steal:** Deal cards preview on homepage, compliance/regulatory trust badges, portfolio performance chart

## 3. Redesign Strategy

### Target Audience
- **Primary:** High-net-worth Indian investors (35-55) seeking alternative assets
- **Secondary:** Startup founders seeking angel/seed funding
- **Tertiary:** NRI investors wanting exposure to Indian startup ecosystem

### Brand Essence (Distilled)
> "The insider's gateway to India's next billion-dollar companies."

**Tone:** Authoritative, data-confident, aspirational, premium but approachable

### Color System (Redesign)
| Token | Value | Usage |
|-------|-------|-------|
| `--primary` | `#1a0a6e` | Deep Indigo (brand) |
| `--primary-mid` | `#2d1ea3` | Medium Indigo |
| `--surface` | `#0d0a2e` | Dark Navy (dark designs) |
| `--accent` | `#f97316` | Orange (CTAs) |
| `--accent-gold` | `#f59e0b` | Gold (premium feel) |
| `--accent-electric` | `#6366f1` | Electric Purple (modern) |
| `--text-primary` | `#ffffff` | White (on dark) |
| `--text-muted` | `rgba(255,255,255,0.6)` | Muted text |
| `--glass` | `rgba(255,255,255,0.05)` | Glassmorphic surfaces |
